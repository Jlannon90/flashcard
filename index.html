JavaScript (JS)

JavaScript (JS) is a programming language used to make web pages interactive.

Operators

An operator is a special character (or characters) that indicates an action to be performed.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name. Always use the var keyword when initializing variables in JavaScript.

Variable naming conventions

Variable naming conventions use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments.

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Some methods take one or more arguments that provide the method with additional information to help it perform its action. An argument is what you pass into a function or method.

Parameters

A parameter is a variable that's assigned to the argument.

Return Value

The return value is the method's response.

Chaining methods

Chaining methods call a method directly on the return value of another method.

Strings

A string is a JavaScript data type that represents the exact text of whatever is enclosed in the quotes. Such as "This is a string!".

Booleans

True and false are booleans. They are not strings - they simply represent being true or false.

Undefined

Undefined is an object (such as a variable) without a defined value.

NaN

0 / 0; returns NaN stands for not a number. NaN is a type of number.

Escape

An escape tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.
Example: "Quote the raven, \"Nevermore.\""

Alert

Alert opens a dialog box and returns undefined.

Comments

A comment is anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

jQuery is a JavaScript library that makes it easier to incorporate JavaScript on your web pages.

Attributes

Attributes are added to HTML elements and used by a jQuery as selectors to control the target system.
